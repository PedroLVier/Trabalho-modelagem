# Sistema de achados e perdidos
Trabalho da matéria Praticas Extensionistas 3, referente a um site para achados e perdidos de uma universidade
O trabalho tem como objetivo o desenvolvimento do modelo de um sistema web de achados e perdidos voltado para o ambiente universitário. A proposta busca solucionar a dificuldade na recuperação de itens perdidos dentro da instituição, por meio de uma plataforma digital que centraliza o cadastro e a busca de objetos.
O sistema permite que usuários, como alunos, professores e funcionários, registrem itens perdidos ou encontrados, realizem buscas e entrem em contato entre si. Além disso, conta com a atuação de um administrador responsável pela moderação e aprovação das publicações, garantindo a organização e confiabilidade das informações.
